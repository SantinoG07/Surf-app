# 🌊 Surf App

This project is part of a school asignament. The creation and development of the app, the improve of skills at those areas of software development.

## 🎯 Goals

Bring the necessary info to everyone that is at the beach, there are two info levels.
The advanced level, this is for experienced surfers, some of the data that is important to prepare a seasson;
The principal level, this one is for the average tourist, reduced data with some about the wheather in the next days.
Also there is a section extra to the tourist, a map with all the places and markets around the world, it isn't a problem where you are vacationing.


## 🔰 Comunity

At the begenning of this project, the main user was the *surfer*. But, the fact that is a "nicho" user, forced to include the tourists.
